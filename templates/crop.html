Crop Prediction model

This model uses random forest classifier for the prediction case. I have trained the model with the dataset provided. The accuracy rate is 98.9%. I have also done some bar plots inside the main file. For the model I preferred to use google collab. In this model the user who can be a farmer or a normal person needs to enter the data i.e. parameters of the soil where the farmer is planning to do farming. The parameters mainly includes soil minerals, rainfall of the place, the temperature, humidity and ph level which is basically the acidity or alkalinity of the soil. Based on the given parameters it will provide you with the appropriate crop to be grown there.
